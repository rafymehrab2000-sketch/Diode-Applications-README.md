# Reflection

In this assignment I learned how to choose a suitable diode for a practical circuit and how to justify the choice using voltage rating, current rating, and availability. I also practiced using LTspice to simulate the basic operation of a diode protection circuit.

The main difficulty was understanding how to set up the simulation correctly and how to connect the diode in the right orientation for protection. After working through the circuit step by step, I understood the purpose of the diode much better.
